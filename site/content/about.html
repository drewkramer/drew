What This Site is Built With

Hugo
Incredibly fast static site builder. Easy to learn and get started, especially if you're familiar with how Wordpress works. Currently using and loving the victor-hugo boilerplate.

Bootstrap
CSS framework that makes creating layouts fast and simple. It's also easily expanded upon per each design.

Sass
CSS on steroids. Variables, nesting, mixins, nuff said. I'm also using a Sass plugin called Shevy to create a modular typography scale. Super handy.

Cloudinary
All media from this site is stored on Cloudinary. Serves images from their fast cdn and they have a lot of tools to make working with media easier, including serving responsive images.

Netlify
Publish static site projects. It's free, and their platform just straight up rules. Highly recommend.

Gulp + npm
Local server development + build automation. I'm still learning the power, but these tools just make things way easier for development.

Google Fonts
I'm using two type faces on this site, Butler and Karla. Karla is a great free font from Principal Design served up by Google Fonts, which is my go-to platform for webfonts.
